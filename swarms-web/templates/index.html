Conversation Modes

Hybrid Mode (Recommended)

Two-phase conversations for rich, multi-layered discussions:

  • Phase 1: Independent thoughts
  • Phase 2: Response round with rebuttals and agreements
All-Speak Mode

Fast-paced group discussions where all motivated agents contribute:

  • Priority-ordered responses
  • Context building between agents
Sequential Mode

Traditional turn-taking with fairness rotation:

  • One speaker per turn
  • Prevents agent monopolization
Pure Priority Mode

Meritocratic discussions where motivation drives participation:

  • Highest motivated agent speaks
  • Natural leader-follower dynamics
